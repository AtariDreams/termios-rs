pub fn tcsendbreak(fd: RawFd, duration: c_int) -> Result<()>

Transmits data to generate a break condition.

If the terminal device is using asynchronous data transmission, tcsendbreak() transmits a continuous stream of zero bits for a specific duration.

Parameters

  • fd should be an open file descriptor associated with a terminal.
  • duration controls the duration of the transmitted zero bits. A value of 0 causes a transmission between 0.25 and 0.5 seconds. A value other than 0 causes a transmission for an implementation-defined period of time.
